## 仿移动端网易云音乐(只涉及歌曲部分)
基于Vue2+VueRouter+Vuex +Nginx
## Project setup
```
npm install
```

## API
[网易云音乐Node版API](https://binaryify.github.io/NeteaseCloudMusicApi/#/?id=neteasecloudmusicapi)

### 小优化
+ 图片使用Vue-LazyLoad实现懒加载
+ 手机号输入只能输入数字
+ 密码不能输入汉字
+ 本地缓存数据,减少请求
